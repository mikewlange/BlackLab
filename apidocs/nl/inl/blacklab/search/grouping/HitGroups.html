Class HitGroups

  • java.lang.Object
    • nl.inl.blacklab.search.grouping.HitGroups
  • All Implemented Interfaces:
    Iterable<HitGroup>, DocOrHitGroups
    Direct Known Subclasses:
    ResultsGrouper

    public abstract class HitGroups
extends Object
implements Iterable<HitGroup>, DocOrHitGroups
    Groups results on the basis of a list of criteria. This class allows random access to the groups, and each group provides random access to the hits. Note that this means that all hits found must be retrieved, which may be unfeasible for large results sets.
