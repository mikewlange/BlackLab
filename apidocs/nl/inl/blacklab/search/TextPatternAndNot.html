    public class TextPatternAndNot
extends TextPattern
    AND query for combining different properties from a complex field. Note that when generating a SpanQuery, the Span start and end are also matched! Therefore only two hits in the same document at the same start and end position will produce a match. This is useful for e.g. selecting adjectives that start with a 'b' (queries on different property (sub)fields that should apply to the same word).
