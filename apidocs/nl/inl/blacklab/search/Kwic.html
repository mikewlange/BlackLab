nl.inl.blacklab.search

Class Kwic

  • java.lang.Object
    • nl.inl.blacklab.search.Kwic

  • public class Kwic
extends Object
    A "keyword in context" for a hit (left context, hit text, right context). The Hits class matches this to the Hit. This object may be converted to a Concordance object (with XML strings) by calling Kwic.toConcordance().
