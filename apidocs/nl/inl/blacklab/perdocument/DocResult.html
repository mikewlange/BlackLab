        public Hits getHits(int max)
        Get some or all hits in this document.
        Parameters:
        max - the maximum number of hits we want, or 0 if we want all hits. Only use 0 if you really want all the hits. For example, if making concordances, it is more efficient to retrieve only some of the hits, so BlackLab won't fetch context for all the hits, even the ones you don't want concordances for.
        Returns:
        the hits
