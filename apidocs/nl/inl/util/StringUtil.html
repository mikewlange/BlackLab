      • getPerWordCollator

        public static RuleBasedCollator getPerWordCollator(Collator base)
        Returns a new collator that takes spaces into account (unlike the default Java collators, which ignore spaces), so we can sort "per word". Example: with the default collator, "cat dog" would be sorted after "catapult" (a after d). With the per-word collator, "cat dog" would be sorted before "catapult" (cat before catapult). NOTE: the base collator must be a RuleBasedCollator, but the argument has type Collator for convenience (not having to explicitly cast when calling)
        Parameters:
        base - the collator to base the per-word collator on.
        Returns:
        the per-word collator
